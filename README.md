## Shiny Side of LLMs blog series

Want to learn more about LLMs? Not only how to use them programmatically in R (or Python), but also how to build interactive web apps with them using Shiny? Check out "The Shiny Side of LLMs" blog series on the Shiny blog:

- Part 1: [What LLMs Actually Do (and What They Don't)](https://shiny.rstudio.com/blog/shiny-side-of-llms-part-1)
- Part 2: [Talking to LLMs: From Prompt to Response](https://shiny.rstudio.com/blog/shiny-side-of-llms-part-2)
- Part 3: [Biuld Your First LLM App with Shiny for Python or R](https://shiny.rstudio.com/blog/shiny-side-of-llms-part-3)

In this series you'll build a full blown app called "DeckCheck", that analyses your Quarto slides and improves them using LLMs.
